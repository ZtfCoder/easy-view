# EasyView - Product Overview

EasyView is a lightweight macOS image viewer built with SwiftUI. It provides a clean, immersive experience for browsing images with minimal UI chrome.

## Core Features

- **Image Browsing**: Open single/multiple images or folders; auto-loads sibling images from same directory
- **Supported Formats**: JPG, JPEG, PNG, GIF, HEIC, WebP, TIFF, BMP
- **Zoom**: Trackpad pinch, Command+scroll wheel, double-click to reset
- **Navigation**: Arrow keys, on-screen buttons, ESC to close
- **Performance**: NSCache-based image caching, async loading, neighbor preloading
- **UX**: Immersive UI with auto-hiding controls, window pinning, drag-to-pan

## Target Platform

- macOS 12.0+
- Native SwiftUI application
- Sandboxed with security-scoped bookmarks for folder access persistence

## Language

- UI and documentation are in Chinese (Simplified)
- Code comments mix Chinese and English
