# 如何将 easyView 设置为默认图片查看器

## 使用方法

### 方式一：右键打开（临时）

1. 在 Finder 中找到图片文件
2. 右键点击 → "打开方式" → "easyView"

### 方式二：设置为默认查看器

1. 右键点击任意图片 → "显示简介"（或 `⌘ + I`）
2. 在"打开方式"下拉菜单中选择 "easyView"
3. 点击"全部更改..."
4. 确认后，所有相同格式的图片都会用 easyView 打开

### 方式三：拖放到 Dock

将图片文件直接拖放到 Dock 中的 easyView 图标上

## 支持的图片格式

- JPEG (`.jpg`, `.jpeg`)
- PNG (`.png`)
- GIF (`.gif`)
- TIFF (`.tiff`, `.tif`)
- BMP (`.bmp`)
- HEIC (`.heic`)
- WebP (`.webp`)
- SVG (`.svg`)

## 应用行为

- 双击图片打开后，会自动加载同文件夹的其他图片
- 可以用左右箭头浏览整个文件夹
- 关闭窗口后应用会退出

## 提示

如果想恢复为系统默认的预览应用，按相同步骤在"打开方式"中选择"预览"并点击"全部更改"。
