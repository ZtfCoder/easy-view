# 如何将 easyView 设置为默认图片查看器

## ✨ 新特性

现在 easyView 支持：
- **不启动空窗口**：应用启动时不显示窗口，等待用户双击图片文件
- **双击图片打开**：在 Finder 中双击任意图片文件，用 easyView 打开
- **文件关联**：可以设置为默认图片查看器

## 📝 使用方法

### 方式一：临时使用（右键打开）

1. 在 Finder 中找到任意图片文件
2. **右键点击图片** → 选择 **"打开方式"** → **"easyView"**
3. 图片会在 easyView 中打开

### 方式二：设置为默认图片查看器

1. 在 Finder 中找到任意图片文件（如 `.jpg` 或 `.png`）
2. **右键点击图片** → 选择 **"显示简介"**（或按 `⌘ + I`）
3. 在简介窗口中找到 **"打开方式"** 部分
4. 点击下拉菜单，选择 **"easyView"**
5. 点击 **"全部更改..."** 按钮
6. 在弹出的确认对话框中点击 **"继续"**

这样，所有相同格式的图片文件双击后都会用 easyView 打开！

### 方式三：从 Dock 拖放

1. 启动 easyView 后，将其固定到 Dock
2. 将图片文件直接拖放到 Dock 中的 easyView 图标上
3. 应用会自动打开该图片

## 🎯 应用行为

### 启动时
- **不显示空窗口**：应用启动后在后台等待
- **Dock 图标会出现**：可以点击 Dock 图标打开文件选择器

### 双击图片文件时
- **自动创建窗口**：显示选中的图片
- **自动获取权限**：有权访问该图片文件
- **单文件模式**：只显示这一张图片（无法切换到同文件夹的其他图片，因为没有文件夹权限）

### 想浏览整个文件夹
如果想查看文件夹中的所有图片（支持左右切换）：
1. 点击左上角的 **文件夹图标** 按钮
2. 在弹出的对话框中选择 **文件夹**（不是单个文件）
3. 这样就能浏览文件夹中的所有图片了

## 🚀 支持的图片格式

- **JPEG** (`.jpg`, `.jpeg`)
- **PNG** (`.png`)
- **GIF** (`.gif`)
- **TIFF** (`.tiff`, `.tif`)
- **BMP** (`.bmp`)
- **HEIC** (`.heic`) - Apple 高效图像格式
- **WebP** (`.webp`)
- **SVG** (`.svg`)

## 💡 提示

- 如果想恢复为系统默认的预览应用（Preview），按照相同步骤在"打开方式"中选择"预览"并点击"全部更改"
- macOS 可能需要重启 Finder 才能完全生效：按住 `Option + 右键点击 Dock 中的 Finder 图标` → 选择"重新启动"
